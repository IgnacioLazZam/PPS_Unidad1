# PPS_Unidad1 - Prácticas de Evaluación

Repositorio para las prácticas de la Unidad 1 de Puesta en Producción Segura.

## Descripción

Este proyecto implementa tres funciones principales en Python:

- **esBinario**: Valida si una cadena es un número binario válido
- **estaEnRango**: Verifica si un valor está dentro de un rango
- **estaEnLista**: Comprueba si un valor está en una lista

## Ejecución de programas
```bash
# Programa binario
python src/functions/binario.py

# Programa lista
python src/functions/lista.py
```

## Ejecución de tests

### Con unittest
```bash
python -m unittest src/tests/test_funciones.py
```

### Con pytest
```bash
python -m pytest src/tests/test_funciones_pytest.py -v
```
